# 💤 Sleep Detection Program

A lightweight Python-based application designed to detect sleep patterns or inactivity.

---

## Steps to run

Follow the steps below to set up and run the program on your machine.

### 1. Install `uv`

This project uses **uv** for environment and dependency management.

```bash
pip install uv
```

---

### 2. Ensure Python 3.10 is Installed

This project is built specifically for **Python 3.10**.

If you don't have it installed, run:

```bash
uv python install 3.10
```

---

### 3. Create a Virtual Environment

Set up a virtual environment using Python 3.10:

```bash
uv venv --python 3.10
```

---

### 4. Run the Application

Execute the main program with:

```bash
uv run main.py
```

---

## Notes

* Using the specified Python version ensures compatibility and avoids unexpected issues.

---

## Purpose

This project is intended for:

* Detecting sleep or inactivity states
* Learning and experimenting with Python-based monitoring systems
* Serving as a base for more advanced tracking or alerting tools
